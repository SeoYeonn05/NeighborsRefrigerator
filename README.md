## 이웃들간 남는 식재료를 공유할 수 있도록 제작한 어플리케이션입니다.

## 제공 기능
 - 자기 위치에 기반한 식재료 나눔, 구함 게시물 표시 기능
 - 글 게시자와 실시간 채팅 기능
 - 불건전한 게시물 신고 기능
 - 나눔 건수에 기반한 사용자 레벨 관리 기능
